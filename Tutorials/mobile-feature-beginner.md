# Beginner Tutorial: Sending Your First Message (Mobile)

## Overview
This tutorial helps new users send their first message
using the mobile application.

**Estimated time:** 5 minutes  
**Audience:** Firstâ€‘time mobile users

---

## Prerequisites
Before you begin, ensure that:
- The mobile application is installed
- You are signed in with a valid account
- Internet connectivity is available

---

## Step 1: Open the Application
1. Locate the application icon on your device.
2. Tap the icon to launch the app.
3. Wait for the home screen to load.

---

## Step 2: Navigate to Chat
1. From the bottom navigation bar, select **Chat**.
2. Review the list of existing conversations.
3. Tap **New Chat** to start a conversation.

---

## Step 3: Select a Contact
1. Search for a user by name or email.
2. Select the contact from the search results.
3. The chat window opens automatically.

---

## Step 4: Send a Message
1. Tap inside the message input field.
2. Enter a short message.
3. Select **Send**.

---

## Verification
- The message appears in the conversation thread.
- A timestamp is displayed below the message.
