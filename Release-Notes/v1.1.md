# Release Notes — Version 1.1

**Release Date:** February 5, 2026

This release introduces usability improvements, performance enhancements,
and stability fixes based on early user feedback.

---

## What's New

- Message pinning within chat conversations
- Improved search for messages and files
- Enhanced notification controls at the user level

---

## Enhancements

- Faster message loading in high‑traffic channels
- Improved accessibility support for keyboard navigation
- Optimized mobile performance on low‑bandwidth networks

---

## Bug Fixes

- Fixed delayed notifications on mobile devices
- Resolved issues with file preview rendering
- Addressed rare application crashes during startup
