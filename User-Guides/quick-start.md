# Quick Start Guide

## Purpose
This guide helps new users start using the application
within a few minutes of installation.

## Step 1: Sign In
1. Launch the application.
2. Enter your email address.
3. Complete authentication if prompted.

## Step 2: Set Up Profile
- Add your display name.
- Upload a profile photo (optional).
- Select your preferred language.

## Step 3: Start Using Core Features
- Open the main dashboard.
- Access available teams or projects.
- Send your first message or action.
