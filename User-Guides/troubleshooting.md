# Troubleshooting Guide

## Overview
This document helps users identify and resolve common issues
encountered while using the application.

## Sign‑In Issues

### Unable to Log In
**Possible causes**
- Incorrect credentials
- Network connectivity issues
- Account not activated

**Resolution**
1. Verify email and password.
2. Check internet connection.
3. Contact the system administrator if the issue persists.

## Application Performance Issues

### Slow Response or Freezing
**Possible causes**
- Poor network connection
- Outdated application version

**Resolution**
1. Refresh the application.
2. Update to the latest version.
3. Restart the device.

## Notification Issues

### Notifications Not Received
**Possible causes**
- Notifications disabled in settings
- Device‑level notification restrictions

**Resolution**
1. Check application notification settings.
2. Enable notifications in device settings.

## When to Contact Support
- Issue persists after troubleshooting steps
- Error messages prevent application use
